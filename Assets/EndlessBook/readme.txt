Thank you for choosing EndlessBook

To get started, select the appropriate package for your project. If you are using the Universal Render Pipeline, select EndlessBook_URP. If you are using the other pipelines, select EndlessBook_Other. Double click the asset to unpackage.

EndlessBook is a complete way to visualize a book inside your Unity scene.

EndlessBook is more than just an animated mesh. It comes complete with scripts to handle updating your pages, animating the book states and page turns, swapping out static models when not animating, and firing handlers when the book changes state or starts / stops a page turn.

​Make your pages come alive with custom shaders and render textures. Offer an interactive experience to your users that sets your project apart!
